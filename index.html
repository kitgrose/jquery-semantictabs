SemanticTabs

A jQuery plugin to make your tabs navigation element semantic

Download .zip Download .tar.gz View on GitHub

Introduction

With this plugin you can make your tabs navigation element semantic following the HTML5 Outline pattern.

$("#mytabset").semantictabs({
  panel:'.panel',         //-- Selector of individual panel body.
  head:'h3',              //-- Selector of element containing panel head.
  active:':first',        //-- Selector of panel to activate by default.
  removeHead: true        //-- Remove headings content from panel.
});
